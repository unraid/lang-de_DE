Process=
Kill Process=Prozess Beenden
Process ID=Prozess ID
Open Files=Geöffnete Dateien
Files that may prevent shutdown=Dateien die ein Herunterfahren möglicherweise verhindern
File Name and Path=Dateiname und Pfad
An entry for bash means you have a terminal session open with the working directory on the array=Ein Bash-Eintrag bedeutet, dass eine Terminalsitzung mit dem Arbeitsverzeichnis auf dem Array geöffnet ist
If smbd is listed, it means that another computer has a samba share mounted=Ein smbd-Eintrag bedeutet, dass auf einem anderen Computer eine Samba-Freigabe geöffnet ist
Files open by Samba usually wont prevent array shutdown=Geöffnete Dateien von Samba verhindern normalerweise nicht das Herunterfahren
No open files=Keine Geöffneten Dateien
Kill=Beenden
Command timed out=Zeitüberschreitung des Befehls
Cannot get list of open files=Fehler beim Laden der Liste der offenen Dateien
